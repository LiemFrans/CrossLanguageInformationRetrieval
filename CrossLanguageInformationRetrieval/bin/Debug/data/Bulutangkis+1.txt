Kvitova Juara Usai Tarung Tiga Set Lawan Bertens--

Madrid - Petra Kvitova tampil sebagai juara Madrid Terbuka. Petenis asal Republik Ceko itu harus berjuang sepanjang tiga set melawan Kiki Bertens di babak final.

Dalam pertandingan di La Caja Magica, Minggu (13/5/2018) dini hari WIB, Kvitova harus bertarung selama dua jam 52 menit untuk menyudahi perlawanan Bertens dengan skor 7-6(8-6), 4-6, 6-3.

Kvitova merebut set pertama yang berlangsung selama 75 menit lewat tie-break. Tapi Bertens bangkit di set kedua.

Di set penentuan, Bertens sempat mematahkan servis Kvitova lebih dulu. Tapi Kvitova balas dengan dua kali mematahkan servis Bertens untuk berbalik unggul 4-2.

Bertens sempat merebut gim berikutnya. Tapi Kvitova langsung membalas dengan memenangi dua gim berikutnya untuk mengunci kemenangan.

Kvitova pun menjadi petenis pertama yang punya tiga titel di Madrid Terbuka. Pemilik dua gelar Wimbledon itu melewati Serena Williams dan Simona Halep yang dua kali juara di Madrid.

"Setiap gelar terasa luar biasa. Menang tiga kali di Madrid, itu berarti sesuatu. Itu bukan hal yang terjadi setiap hari, jadi saya bangga kepada diri saya sendiri," ucap Kvitova seperti dikutip situs resmi WTA.
